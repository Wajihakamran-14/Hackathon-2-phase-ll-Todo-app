# Frontend Skill

## Instructions

1. **Page & Layout Structure**
   - Create responsive page layouts
   - Use grid or flexbox for alignment
   - Define header, footer, main content, and sections
   - Ensure full viewport height and mobile-first design

2. **Components**
   - Break UI into reusable components
   - Pass data via props or state
   - Handle component lifecycle and events
   - Maintain consistent styling across components

3. **Styling**
   - Use CSS, SCSS, or utility-first frameworks like Tailwind
   - Apply spacing, colors, and typography consistently
   - Use CSS variables for maintainability
   - Ensure accessibility (contrast, focus states, ARIA attributes)

4. **Visual Elements**
   - Add images, icons, and illustrations
   - Apply backgrounds, gradients, and overlays
   - Use buttons, cards, modals, and forms
   - Ensure interactive elements provide feedback

5. **Animations & Interactions**
   - Fade, slide, and scale effects
   - Parallax scrolling or hover effects
   - Smooth transitions between pages or components
   - Consider performance and accessibility

## Best Practices
- Keep component code clean and reusable
- Follow consistent naming conventions
- Prioritize mobile-first and responsive design
- Optimize images and assets
- Use semantic HTML for accessibility
- Test across browsers and devices
